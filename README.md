# 🌐 Cristall P2P Relay Server

Relay-сервер для децентрализованной сети Кристалл.

## 🚀 Запуск на GitHub Codespaces:

1. Открыть Codespace
2. Установить зависимости: `npm install`
3. Запустить сервер: `npm start`
4. Открыть порт 7777 как Public
5. Подключение узлов Кристалл

## 📋 Файлы:

- **cristall-p2p.cjs** - основной P2P модуль (362 строки)
- **relay-server.cjs** - скрипт запуска relay (45 строк)
- **package.json** - зависимости Node.js

## 🔧 Архитектура:

- **P2P Protocol:** WebSocket
- **Port:** 7777
- **Mode:** Relay (пересылка сообщений между узлами)
- **Transport:** TCP/WebSocket с автоматическим heartbeat

## 📡 Подключение:

Из узлов Кристалл подключайтесь к:
```
wss://ВАШ_АДРЕС.preview.app.github.dev
```

## 📊 Мониторинг:

Relay логирует все подключения и статистику каждые 30 секунд.

## 🔒 Безопасность:

В текущей версии нет аутентификации. Для production добавить:
- Token-based auth
- Whitelist узлов
- Rate limiting
